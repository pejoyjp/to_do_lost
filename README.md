# tauri app

基于 Tauri + React + Ts 构建的精美 todo app

## 功能

1. todo
    1. 增加
    2. 删除
    3. 拖拽排序
    4. 编辑
    5. 过滤状态
2. 主题
    1. 浅色
    2. 深色
    3. 跟随系统
3. 窗口置顶
4. 磨砂透明背景(win11,macos)
5. 数据缓存
